# PR_7 NAVIGATOR

A React + Vite based Employee Management Navigator project.

## 🚀 Project Description

PR_7 NAVIGATOR is a simple employee management interface built using
React.
It allows users to add employees and view employee details in a
structured interface.

This project demonstrates basic **React concepts, routing, and component
structure**.

## 🛠️ Technologies Used

-   React
-   Vite
-   JavaScript (ES6)
-   Bootstrap
-   React Router DOM
-   React Icons

## 📂 Project Structure

    src
     ├── assets
     ├── components
     │    └── Header.jsx
     ├── pages
     │    ├── Add_employee.jsx
     │    └── View_employees.jsx
     ├── App.jsx
     └── main.jsx

## ⚙️ Installation

Clone the repository:

    git clone https://github.com/your-username/PR_7_NAVIGATOR.git

Go to project folder:

    cd PR_7_NAVIGATOR

Install dependencies:

    npm install

Run the development server:

    npm run dev
